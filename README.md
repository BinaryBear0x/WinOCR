# WinOCR

WinOCR, görsel dosyalar veya panodaki resimler üzerinden OCR işlemi yaparak metin çıkarmayı hedefleyen Windows Forms uygulamasıdır.

## Ne Yapar

- dosyadan görsel yükler
- panodaki resmi okuyabilir
- Tesseract ile metni çıkarır
- sonucu arayüzde gösterir

## Teknolojiler

- C#
- Windows Forms
- Tesseract OCR

## Notlar

- Uygulamanın çalışması için uygun `tessdata` dosyalarının uygulama dizininde bulunması gerekir
